= Container Image Signing and Attestation

This lab teaches cryptographic image signing and SBOM attestation using cosign,
establishing a verifiable chain of trust for container images.

== Lab Overview

Participants verify Red Hat's published image signatures, then sign their own images
using a local registry, generate cosign key pairs, and attach signed SBOM attestations.

== Modules

* Module 01: Verify Published Signatures -- inspect Red Hat's signing chain
* Module 02: Push to Registry and Generate Keys -- prepare for signing
* Module 03: Sign and Verify -- cryptographic image signing
* Module 04: SBOM Attestation -- attach and verify signed SBOMs

== Infrastructure

Requires ZeroSSL credentials in config/secrets.yaml for the SSL-enabled local registry.

== Duration

18-22 minutes
